This code is used to back up virtual machines (VMs) stored in the Parallels directory. It will create a tar.gz archive of each VM and copy it to the destination directory.

## Requirements

- Python 3
- Access to the Parallels directory and the destination directory

## Running the Code

1. Open the code in a text editor and modify the `SOURCE` and `DESTINATION` variables to the correct paths.
2. Run the code using the command `python3 <filename>`

## Options

No options are available for this code.
